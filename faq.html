1. CompuCell3D crashes on Windows with the following information: Qt has caught an exception thrown from an event handler. Throwing exceptions from an event handler is not supported in Qt. You must reimplement QApplication::notify() and catch all exceptions there. This application has requested the Runtime to terminate it in an unusual way. Please contact the application's support team for more information.

Answer: Try closing CC3D and removing configuration file where CC3D stores its settings. The file location is- your_home_directory - usually c:\users\yourname>\AppData\Roaming\Biocomplexity\cc3d_default.ini.

After you remove this file run CC3D again.

2. CompuCell3D does not work after update?

Answer: There might be issue while removing old CompuCell3D version while update. In case you can't use CompuCell3D after update, please manually delete CompuCell3D directory which usually at - C:\CompuCell3D for windows. Once directory is removed try reinstalling CompuCell3D.
