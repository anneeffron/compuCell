CompuCell3D Source Code

Access to the main CompuCell3D development source code and developer documentation repository: https://github.com/CompuCell3D/CompuCell3D

To download a local copy of the CompuCell 3D source code repository, read-only access is available from github.com . For example, on Linux/Unix/Mac OS X systems:

mkdir CC3D_GIT cd CC3D_GIT/ git clone https://github.com/CompuCell3D/CompuCell3D.git .
