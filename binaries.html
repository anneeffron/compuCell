Installation Issue?

If you are facing any problem with installation please contact CompuCell3D developer Jim at - jsluka@iu.edu, or report any bugs on our Q&Apage: CC3D - AllAnswered.

Usually, it is an minor issue that can be solved easily.

***NEW*** CC3D Version 3.7.8

New Features

  • Python parameter Steering via Interactive Sliders
  • Multiple Bug Fixes

IMPORTANT: This is a release candidate. It is fully functional new version of CompuCell3D. Before making it an official release we want to give users an opportunity to try it on their machines and report any bugs on our Q&A page: CC3D - AllAnswered

Binary distribution packages:

Windows - 32-bit and 64-bit

IMPORTANT: If after installation of new CC3D you encounter problems on Windows. Please manually remove the directory where CC3D is installed and try using installer again. This is a one-time inconvenience associated with the transition to the new ways of storing Player settings. You may also need to remove _settings.sqlite file in the Simulation subfolder of the cc3d project folder
